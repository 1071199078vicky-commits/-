# 1071199078vicky-commits

This repository contains a Next.js 14 starter configured with TypeScript, Tailwind CSS, and shadcn/ui. It includes a chart playground page powered by Recharts for visualizing numeric inputs.

## Getting started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## Scripts
- `npm run dev` – start the Next.js development server
- `npm run build` – create an optimized production build
- `npm run start` – start the production server
- `npm run lint` – lint the project with ESLint
